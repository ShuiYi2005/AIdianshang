# 简历用 — AI 智能客服系统项目描述

---

## 简短版（一句话）

基于 Dify 自托管平台 + DeepSeek API 搭建 AI 智能客服系统，具备知识库问答、订单查询、物流跟踪、人工兜底等功能，目标接入抖店飞鸽客服系统。

---

## 详细版（简历项目经历用）

### AI 智能客服系统 | Dify / Docker / DeepSeek API / n8n

**项目背景：** 为电商店铺搭建 7×24 小时 AI 智能客服，自动回复常见商品咨询，查询订单/库存/物流信息，敏感场景自动转人工。

**核心职责与成果：**

**1. 技术选型与架构设计**
- 选定 Dify（自托管 LLMOps 平台）+ DeepSeek API + n8n（工作流）技术路线
- 设计 6 容器微服务架构：PostgreSQL + Redis + Weaviate + Dify API/Worker/Web
- 使用 Docker Compose 实现一键部署

**2. 攻克网络限制下的部署难题**
- Dify 官方 Docker 镜像在国内镜像源全部被墙（403 Forbidden），直连 Docker Hub 超时
- 通过版本回溯策略，成功部署 Dify 0.15.3（旧版不依赖 plugin-daemon 插件服务）
- 解决旧版本存储系统（Apache OpenDAL）配置兼容性问题

**3. 系统集成与功能实现**
- 集成 DeepSeek API 作为推理引擎，支持多轮对话与 Function Calling
- 对接 Weaviate 向量数据库实现知识库 RAG 检索
- 搭建 db-simulator 模拟数据服务，验证订单/库存/物流查询链路
- 配置 n8n 工作流引擎，为抖店对接做好准备

**4. 抖店开放平台预研**
- 完成飞鸽客服系统 API 对接方案调研
- 识别出 API Key 申请需要营业执照资质

**技术栈：** Dify 0.15.3 / Docker Compose / PostgreSQL / Redis / Weaviate / DeepSeek API / n8n / Python / FastAPI
